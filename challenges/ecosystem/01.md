## üëæ Before we start the exercise

There is 2 ways to install extentions on VSCode.

### The manual way:

1. Launch VSCode Quick Open

- macOS (`cmd` + `shift` + `p`)
- windows (`ctrl` + `shift` + `p`)

2. Type ">install extensions"
3. Search the name
4. Click on "install"

### The automated way:

- [ ] open your terminal and run

```console
code --install-extension dbaeumer.vscode-eslint esbenp.prettier-vscode
```

### Turbo `console.log()`
